## What Is Cursor?

Cursor is a VS Code fork that replaces traditional autocomplete with an AI agent. It indexes your entire project and lets you describe changes in natural language. The killer feature is **Composer** — tell it "add user authentication with JWT" and it creates the middleware, routes, and tests across files.

## What Is GitHub Copilot?

Copilot is Microsoft's AI pair programmer. It lives inside your existing IDE and provides ghost-text suggestions as you type. In 2026, it also has **Agent mode** for multi-step tasks and **Code Review** for PR analysis. It's designed to be invisible — you code, and suggestions appear.
